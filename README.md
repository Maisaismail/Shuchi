Project Description:Shuchi (ശുചി)

Shuchi is a specialized waste management web application designed to digitize and streamline the operations of the Haritha Karma Sena (Green Action Force). The platform serves as a bridge between local residents and waste collection workers, focusing on improving the efficiency of house-to-house collection tracking within Kerala's municipalities.

Tech Stack:
Frontend: React.js / HTML5 & CSS3 (Futuristic Deep Green UI)
Backend: Node.js with Express.js

Key Features:
1.Worker Dashboard (Haritha Karma Sena): A command center for workers to view daily collection lists, filter by "Collected" or "Pending" status, and access resident contact details.

2.Resident Registration System: Secure signup using specific Local Body data (e.g., Kochi Municipality), House Number, and Warden information.

3.Real-time Collection Status: A dynamic database that updates instantly when a worker marks a household as "Collected," visible to municipal supervisors.

4.Resident Profile Hub: A dedicated page for citizens to verify their assigned House Warden, view their collection history, and update ownership details.

System Architecture:

The application utilizes a Role-Based Access Control (RBAC) architecture. The backend API communicates with a MongoDB cluster to store household data, while the frontend provides distinct views: a data-entry-heavy dashboard for workers and a simplified information portal for residents.
